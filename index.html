UX/UI DESIGN

완자 : 완벽한 자취생활의 모든 것

Wanja : everything for living alone perfectly

사용 프로그램 : Photoshop, Illustrator

제작시기 : 2017.11

완자는 자취 생활에 필요한 여러 가지 꿀팁 공유를 목적으로 만들어졌으며, 꿀팁을 보관하고 공유하는 것 뿐만아니라, 사용자들간의 자유로운 정보 공유도 가능합니다. 플랫한 디자인과 아이콘을 통하여 심플함을 강조하였고, UI가 한눈에 들어오도록 구성하였습니다. '꿀팁'에서 '꿀'과 '벌'을 연상하여 귀여운 꿀벌을 어플의 캐릭터로 사용하였고, 노란색과 주황색을 메인 컬러로 사용하여 친근감과 따듯한 느낌 주었습니다. 많은 컨텐츠로 구성 되어 있지만 설명 위주의 컨텐츠가 아닌 이미지 위주의 컨텐츠로 사용자들의 흥미를 유도하며, 설명글은 간략하게 요약하여 지루하지 않도록 구성하였습니다.
